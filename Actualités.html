Réseau

Pour l’instant le jeu multijoueur fonctionne parfaitement localement. Lors de la réalisation du multijoueur j’ai eu pas mal de problème : - Le premier est dû au inpout, en effet lorsque l’on appuyer sur une touche tous les joueurs bougées comme si mon clavier allait contrôler tous les joueurs. Pour le régler j’ai du créé un script PlayerSetup qui vas désactiver les scripts PlayerController, PlayerMotor ainsi que la caméra et l’Audio Listener de tous les autres joueurs pour ne plus les contrôler. Ce script désactive aussi le script PlayerShoot mais aussi le réticule qui apparait lorsque l’on lance le jeu. J’ai ensuite ajouté un script PlayerShoot qui vas utiliser des Raycaste pour aller les autres joueurs. Je l’ai programmé pour qu’il ne puisse que toucher des Player et également à travers les autres joueurs. En modifiant les Layer des joueurs.
